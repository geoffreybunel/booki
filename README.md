# Booki

A responsive homepage for a fictional travel agency.

This project focuses on building a clean and structured layout using semantic HTML and modern CSS techniques.

## Features

- Responsive design (desktop, tablet, mobile)
- Accommodation and activities sections
- Search bar UI
- Semantic and accessible HTML structure

## Tech Stack

- HTML5
- CSS3
- Flexbox
- Media Queries

## Getting Started

Clone the repository:

git clone https://github.com/yourusername/booki.git

Then open the `index.html` file in your browser.
